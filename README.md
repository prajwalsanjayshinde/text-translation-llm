# Text Translation LLM

This project implements a text translation application using a language model. It allows users to translate text into different languages through a simple interface.

## Project Structure

```
text-translation-llm
├── src
│   ├── main.py          # Entry point for the application
│   ├── translator.py    # Contains the Translator class for text translation
│   └── utils.py         # Utility functions for text preprocessing and postprocessing
├── requirements.txt     # Lists the project dependencies
├── README.md            # Project documentation
└── tests
    └── test_translator.py # Unit tests for the Translator class
```

## Installation

To set up the project, clone the repository and install the required dependencies:

```bash
git clone <repository-url>
cd text-translation-llm
pip install -r requirements.txt
```

## Usage

To run the application, execute the following command:

```bash
python src/main.py
```

You will be prompted to enter the text you wish to translate and the target language.
